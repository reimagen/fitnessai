# **App Name**: FitnessAI

## Core Features:

- User Profiles: User authentication and profile management.
- Workout Logging: Log individual workout sessions with exercise details, sets, reps, and weight.
- Progress Tracking: Visualize workout progress and personal stats through charts and graphs.
- AI Recommendation Engine: AI powered tool that recommends personalized workout routines and exercise modifications based on user data.
- Goal Setting: Allow users to define and track fitness goals (e.g., weight loss, muscle gain).
- Screenshot Parsing: Allow users to upload a screenshot to parse workout data.
- Manual Data Entry/Upload: Allow users to enter data manually via a form or upload of an xls/csv file.
- Pages: The app should have 5 pages: 1. home page 2. analysis 3. plan 4. history 5. profile
- Bottom Navigation Bar: There should be a navigation bar with associated icons along the bottom of each page

## Style Guidelines:

- Primary color: A muted blue (#5DADE2) to evoke a sense of calm, focus, and reliability, aligning with the app's goal of guiding users through their fitness journey.
- Background color: A light gray (#F5F5F5), providing a clean and neutral backdrop that doesn't distract from the content.
- Accent color: A vibrant orange (#FFB347) to highlight key actions, goals, and recommendations, injecting energy and enthusiasm into the user experience.
- Body and headline font: 'PT Sans' (sans-serif) to maintain readability and a modern, accessible aesthetic.
- Use a set of minimalist and consistent icons to represent workout types, exercises, and stats. Icons should be clear and easily recognizable.
- Use a clean, card-based layout to organize information. The layout should be responsive and adapt to different screen sizes.
